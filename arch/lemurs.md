# Lemurs

login manager

- github: [lemurs](https://github.com/coastalwhite/lemurs)

## Install

```bash
sudo pacman -S lemurs
```

```bash
# sudo systemctl disable display-manager.service
sudo systemctl enable lemurs.service
```

## Usage

### After scripts

```bash
cd /etc/lemurs/wayland
```

## Configuration

```bash
vi /etc/lemurs/config.toml
```
